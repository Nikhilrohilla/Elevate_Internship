# Calculator CLI Application

## Description
A simple command-line calculator developed in Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Runs continuously until Exit is selected

## Requirements
- Python 3.x

## Run

python calculator.py

# Console-Based To-Do List Application

## Description
A simple command-line To-Do List application developed using Python.

## Features

- Add Tasks
- View Tasks
- Remove Tasks
- Automatically saves tasks
- Loads tasks when the application starts

## Requirements

Python 3.x

## Run

Windows:
python todo.py

macOS/Linux:
python3 todo.py
